# Documentación del Proyecto

Este repositorio contiene la documentación y recursos relacionados con el proyecto de Arquitectura de Software. A continuación, se detalla la estructura de la documentación:

## Diagramas de Clase
- [Backend](Documentacion-proyecto-final/DiagramasDeClase/Backend): Diagramas de clase relacionados con la arquitectura del backend.
- [Frontend](Documentacion-proyecto-final/DiagramasDeClase/Frontend): Diagramas de clase relacionados con la arquitectura del frontend.

## Diagramas de Componentes
- [Backend](Documentacion-proyecto-final/DiagramasDeComponentes/Backend): Diagramas que describen los componentes principales del backend.

## Diagramas de Secuencia
- [Backend](Documentacion-proyecto-final/DiagramasDeSecuencia/BackEnd): Diagramas de secuencia que muestran el flujo de interacción en el backend.
- [Frontend](Documentacion-proyecto-final/DiagramasDeSecuencia/FrontEnd): Diagramas de secuencia que muestran el flujo de interacción en el frontend.

---

# Estructura del Repositorio

- **ADAs/**: Contiene actividades de aprendizaje relacionadas con principios de diseño, patrones arquitectónicos, y más.
- **Assets/**: Recursos adicionales como imágenes y otros archivos multimedia.
- **Backend/**: Código fuente y documentación del backend del proyecto.
- **Frontend**: Código fuente del frontend del proyecto (disponible en la rama "Frontend" del repositorio).
- **Documentacion-proyecto-final/**: Contiene diagramas y documentación técnica del proyecto final.
